## ⚡ 简介 introduction

前后端分离的个人博客项目

#### 前端

博客前台基于 vue3、element plus、pinia、axios、vue-router、vite、vue-use、npm、scss、tailwind.css 等主流技术

前端线上预览地址：<http://czjy.fun>
#### 后端

博客后端基于node.js、koa、koa-router、koa-body、seqlize、mysql、minio、bcryptjs、nodemon等主流技术
